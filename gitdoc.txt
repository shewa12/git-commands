#initialize git 
*git init (it will create a empty .git repo. it is basically main git repo)

#remove local git repo
*rm -rf .git

#add specific file
*git add <file name>

#add all file from current folder. (.) dot means current folder
git add . (now files belongs to staging area)

#commit changes. 
*git commit (this command will open integreted text editor with git & it will allow to write multiline messages)

git commit -m "commit message" (commit changes with single line message)

#check git commit detail with author date & time
*git log

#check difference between commits
*git show

#check which files git has been tracking
*git ls-files

#remove specific file from working directory
*rm filename.ext (git bash command)
